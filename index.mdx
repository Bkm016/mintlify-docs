## 阅读路线

- **入门**：从《快速开始》和《命令行与 REPL》开始。
- **工具链**：了解 VS Code 扩展、函数目录生成以及 Mintlify 文档协作方式。
- **语言参考**：根据需要跳转到各语法章节，内容源自 `SYNTAX.md` 的拆分版。
- **运行时/嵌入**：查看解析器、运行时、JSR-223 以及 Java 集成等页面。
- **开发者指南**：贡献流程、测试策略、发布机制与常见故障排查。
- **附录**：CLI 参数、函数目录、术语表等参考资料。
