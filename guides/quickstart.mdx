---
title: "快速开始"
description: "在 JVM 中跑通第一个 Fluxon 脚本"
---

Fluxon 的标准入口是 `org.tabooproject.fluxon.Fluxon`。你可以在 JVM 应用中直接执行表达式或脚本。需要时，
也可以将脚本编译为可复用的字节码。

## 前提

- **JDK**：8 或以上版本。
- **构建工具**：Gradle 或 Maven。
- **发布仓库**：[releases](https://repo.tabooproject.org/repository/releases/)

## 添加依赖（Gradle Kotlin DSL）

```kotlin
repositories {
    mavenCentral()
    maven { url = uri("https://repo.tabooproject.org/repository/releases/") }
}

dependencies {
    implementation("org.tabooproject.fluxon:fluxon-core:1.4.3")
}
```

可选模块（按需添加）：

- `org.tabooproject.fluxon:fluxon-core-jsr223:1.4.3`（JSR223 脚本引擎）
- `org.tabooproject.fluxon:fluxon-core-console:1.4.3`（REPL/控制台）

## 执行第一个脚本（Fluxon.eval）

`Fluxon.eval` 会解析并执行脚本，返回最后一个表达式的值：

```java
import org.tabooproject.fluxon.Fluxon;

public class FluxonHello {
    public static void main(String[] args) {
        Object result = Fluxon.eval("print(\"Hello World\")");
        System.out.println("script returned: " + result);
    }
}
```

## 带环境执行（共享宿主变量）

当你希望脚本能读取/修改宿主预置的变量时，可以显式创建 `Environment`：

```java
import org.tabooproject.fluxon.Fluxon;
import org.tabooproject.fluxon.runtime.Environment;
import org.tabooproject.fluxon.runtime.FluxonRuntime;

public class FluxonHello {
    public static void main(String[] args) {
        Environment env = FluxonRuntime.getInstance().newEnvironment();
        env.defineRootVariable("greeting", "Hi");
        Fluxon.eval("print(&greeting + \", Fluxon!\")", env);
    }
}
```

`Fluxon.compile` 会将脚本编译为可执行的 Java 类，适合做缓存与重复执行；如果你只是想快速跑通脚本，先使用 `eval` 即可。
